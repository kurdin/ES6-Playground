# ES6-Playground
Trying out some of the ES6 features in JS.

### Features explored (at least in a basic level)
- [x] `let` and `const`
- [x] template literals
- [x] basic destructuring
- [x] `for ... of`
- [x] default, rest and spread
- [x] `map` and `set`
- [ ] `weakmap` and `weakset`
- [x] classes and modules
- [x] fat arrow functions
- [x] generators
- [ ] promises
- [ ] `await` and `async`
