# ES6-Playground
Trying out some of the ES6 features in JS.

### Features explored (atleast in a basic level)
- [x] `let` and `const`
- [x] template literals
- [ ] destructuring
- [x] `for ... of`
- [x] default, rest and spread
- [ ] map, set, weakmap and weakset
- [x] classes and modules
- [x] fat arrow functions
- [x] generators
- [ ] promises
- [ ] `await` and `async`
